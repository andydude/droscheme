gosc

The gosc command is the Gos => Go compiler. It is currently only intended to build the standard libraries for the interpreter. It is not recommended to use gosc for other purposes. This command is mostly written in Scheme. The i2s subcommand is written in Guile. The gos2go subcommand is written in Racket. The go2gos subcommand is written in Go.

  • Read the Gos language specification.
  • Read the gosc documentation.
  • Read the gos2go documentation.
  • Read the go2gos documentation.
  • Browse the source code.

Eventually, this logic will be migrated to Droscheme, which shouldn't be too hard since both Guile and Racket are Schemes, and the other command is written in Go, which is the target language. If or when this rewrite is done, then this command will be renamed to 'dsc'.
