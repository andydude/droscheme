    Welcome to Droscheme.

    Droscheme is a variant of the algorithmic language Scheme, which is written in Go. It aims to be fully compliant to both R6RS and R7RS (when it is finished) standards.

    Standard Compliance

    It is currently mostly compliant to R5RS and mostly compliant ro R7RS, but nowhere near R6RS compliant.
